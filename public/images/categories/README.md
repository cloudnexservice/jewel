# Category Images Guide

Add your jewelry product images to this folder. The site will automatically load them for the category cards on the homepage.

## Required Images

Place these images in this directory (`public/images/categories/`):

1. **rings.jpg** - Ring product image (recommended: 600x800px)
2. **bangles.jpg** - Bangles/bracelets product image (600x800px)
3. **necklaces.jpg** - Necklace product image (600x800px)
4. **earrings.jpg** - Earrings product image (600x800px)
5. **bracelets.jpg** - Bracelets product image (600x800px)
6. **pendants.jpg** - Pendant product image (600x800px)

## Image Requirements

- **Format**: JPG, PNG, or WebP
- **Dimensions**: Minimum 600x800px (portrait orientation recommended)
- **Quality**: High resolution for luxury aesthetic
- **Content**: Clear product shots of actual jewelry pieces

## File Naming

Must match exactly (case-sensitive):
- `rings.jpg`
- `bangles.jpg`
- `necklaces.jpg`
- `earrings.jpg`
- `bracelets.jpg`
- `pendants.jpg`

## Example

If you add `rings.jpg` to this folder, the Rings category card will automatically display it when you refresh the page.

No code changes needed—just add your images and they'll appear!
